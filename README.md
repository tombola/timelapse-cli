# Webcam timelapse


## Installation

Works on a mac, apologies to users of other platforms.

Install imagesnap

`brew install imagesnap`

Install [ffmpeg][ffmpeg]. Can take some time.. This is optional, for combining resulting images to video.

`brew install ffmpeg`
